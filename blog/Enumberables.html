Alright. Enumberables. What are they? They're basically methods that are so commonly used in that ruby developers simply decided to add them into array and hash classes. Each is one enumberable method that is commonly used, with the common modifier each_with_index. An example of using each is as follows:

> [1,2,3].each { |num| print "#{num}! " } 1! 2! 3! => [1,2,3]

Another commonly used enumberable is the select enumerable. The select enumberable, when used in an array, returns an array with all elements that meet your criteria. For instance, if I wanted to find all the even numbers in an array, I would use this:

> my_array.select{|item| item%2==0 } => [2,4,6,8,100]

There are a lot of enumerables included in the ruby package, and unfortunately, I haven't been able to go through them all. However, they're all very powerful and useful when used correctly, so do as I plan to and look through them all so you can master them and work smart, not hard!
